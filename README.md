# GE Routes

A web app for browsing and filtering profitable Grand Exchange money-making routes in Old School RuneScape.

## Features

- Browse activities with profit/hr, requirements, category, and intensity
- Filter by minimum GP/hr, category, and intensity
- Look up a player by username to check eligibility for each activity
- Hide activities you don't meet the requirements for
- Sort by any column

## Stack

- React
- Vite
- CSS custom properties for theming

## Running locally

```bash
npm install
npm run dev
```
